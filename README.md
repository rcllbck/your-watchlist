# 🎌 Your Watchlist

App untuk mencatat anime, series, dan movie yang sudah ditonton atau ingin ditonton.

## ✨ Fitur
- Tambah & hapus list anime, series, movie
- Tab Watched & Plan
- Fetch poster otomatis
- Edit info & sub-items
- Dark / Light theme
- Undo & Redo
- Export & Import data
